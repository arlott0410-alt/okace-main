# แนวทางจัดหน้าที่อย่างมืออาชีพ (จัดหน้าที่)

## ระบบยึดจากอะไร

- **แผนก + กะ + วันที่** — การจัดหน้าที่เป็น **รายวัน** ต่อแผนกและกะ
- **รายการหน้าที่ (duty_roles)** — กำหนดต่อแผนก (เช่น หัวหน้างาน, ฝาก-ถอน, แชท)
- **โควต้าคนที่มาทำงานวันนั้น** — ระบบจะใช้เฉพาะคนที่ **มีในตารางกะ (monthly_roster)** ในวันนั้น และ **ไม่หยุด/ไม่ลา (approved/pending)** เป็นกลุ่มสำหรับจัดหน้าที่อัตโนมัติ

---

## วิธีจัดหน้าที่อย่างมืออาชีพในแต่ละวัน

### 1. เตรียมข้อมูลให้ครบ

- **ตารางกะรายเดือน (monthly_roster)** ต้องมีคนในแผนก+กะในวันนั้นแล้ว (จากหน้าตารางงาน / ย้ายกะ)
- **วันหยุด/การลา (holidays)** อนุมัติแล้วจะถูกตัดออกจากกลุ่ม "มาทำงานวันนี้" อัตโนมัติ
- **รายการหน้าที่ (ต่อแผนก)** ต้องสร้างไว้แล้วในบล็อก "รายการหน้าที่ (ต่อแผนก)" เช่น หัวหน้า, ฝาก-ถอน, แชท

### 2. จัดอัตโนมัติทุกวัน (หนึ่งคลิก)

- เปิดหน้า **จัดหน้าที่** → เลือก **แผนก** → **กะ** → **วันที่**
- กดปุ่ม **"สุ่มจัด"** (หรือ "จัดอัตโนมัติวันนี้")
- ระบบจะ:
  - นับคนที่ **มาทำงานวันนั้น** = คนที่มีในตารางกะวันนี้ ลบคนที่หยุด/ลาวันนั้น
  - แสดงข้อความ **"วันนี้มาทำงาน X คน"** ให้เห็นโควต้า
  - สุ่มจัดเฉพาะคนที่มาทำงานวันนั้นเข้าไปในแต่ละหน้าที่ (ถ้าไม่มีตารางกะจะใช้รายชื่อทั้งหมดเป็น fallback)

### 3. ให้พนักงานดูง่าย เข้าใจง่าย

- **พนักงาน (instructor/staff):** เปิดหน้าจัดหน้าที่ → เลือกแผนก/กะ/วันที่ (หรือระบบจำค่าเดิม) → จะเห็น **"หน้าที่ของคุณในวันนี้: [ชื่อหน้าที่]"** ชัดเจน
- **หัวหน้า/แอดมิน:** เห็นทั้งรายชื่อคนมาทำงานวันนี้ และคนที่ไม่มาทำงาน (แยกกลุ่ม) เพื่อให้เข้าใจโควต้าและสามารถลากเปลี่ยนได้ถ้าต้องการ

### 4. ทำซ้ำทุกวัน (ทำให้เป็นนิสัย)

- **ทุกวัน (หรือทุกกะ):** เปิดจัดหน้าที่ → เลือกวันที่วันนี้ → กด "สุ่มจัด" ครั้งเดียว
- ไม่ต้องรัน cron บนเซิร์ฟเวอร์: แค่ให้หัวหน้าหรือแอดมินกดหนึ่งครั้งเมื่อเริ่มกะ หรือตั้งเป็น routine ตอนเช้า

---

## สรุป

| หัวข้อ | คำอธิบาย |
|--------|----------|
| โควต้าคนมาทำงาน | จาก **ตารางกะ (monthly_roster)** ลบคนที่ **หยุด/ลา (holidays ที่ approved/pending)** ในวันนั้น |
| อัตโนมัติทุกวัน | กดปุ่ม **"สุ่มจัด"** ครั้งเดียวต่อวัน (หรือต่อกะ) — จัดเฉพาะคนที่มาทำงานวันนั้น |
| พนักงานดูง่าย | แสดง "วันนี้มาทำงาน X คน" และ "หน้าที่ของคุณในวันนี้: [ชื่อ]" ให้เข้าใจทันที |
